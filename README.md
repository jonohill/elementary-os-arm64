# elementary-os-arm64

elementary OS is officially available only for amd64, and for some specific arm64 devices. This repo produces a generic arm64 iso which should work on other arm64 devices, e.g. on an ARM Mac in a VM.

Download the latest from the [releases page](https://github.com/hillnz/elementary-os-arm64/releases).

## Disclaimer

The produced image is unofficial and untested. The repo owner does not use elementary OS day to day, so it's likely that anything broken won't be discovered.
